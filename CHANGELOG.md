# 更新日志

**作者**: 泥人传说  
**项目地址**: https://github.com/cyberxsboy/WP-Cloudflare-Static-Push

所有重要的项目更改都将记录在此文件中。

## [1.0.0] - 2025-11-06

### 新增
- 首次发布
- 支持 Cloudflare Pages 部署
- 支持 Cloudflare Workers 部署
- 5步引导式设置向导
- 自动推送功能（发布/更新时）
- 手动推送功能（单篇/批量/全站）
- 编辑器快捷推送按钮
- 静态 HTML 生成
- RSS Feed 生成
- Sitemap 生成
- **广告位管理功能**
  - 支持 Google AdSense
  - 支持自定义 HTML 广告
  - 支持 JavaScript 广告代码
  - 5个广告位置（顶部、底部、侧边栏、内容前、首段后）
  - 自动插入广告到静态页面
- 推送日志记录
- 推送历史查看
- 现代化管理界面
- 实时推送状态显示
- 详细错误提示

### 技术实现
- WordPress 6.0+ 兼容性
- PHP 7.4-8.2 兼容性
- Cloudflare API v4 集成
- 安全的 API Token 存储
- Nonce 验证和权限检查
- 模块化代码架构

### 文档
- 完整的 README 说明
- 详细的安装指南
- 使用示例文档
